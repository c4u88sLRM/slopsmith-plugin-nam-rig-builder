# Bender Deluxe — Fender '57 Deluxe (5E3 tweed)

| | |
|---|---|
| **RS gear** | `Amp_TW26` ("Bender Deluxe") |
| **Codename / dir** | `tw26` · `vst/src/amps/tw26/` |
| **Parodia** | Bender Deluxe (sin marca real en la cara) |
| **Real** | **Fender '57 Custom Deluxe** = reissue del **5E3 tweed Deluxe** |
| **Esquemático** | `amps/Fender Deluxe (TW26)/Fender-57-Deluxe-Schematic.pdf` (Fender Musical Instruments, Corona CA — Service Manual, 1 hoja de circuito) |
| **DSP** | `TW26Core.h` (+ `TW26Params.h`, `TW26Plugin.cpp`) |
| **Binario/bundle** | `vst/amps/TW26.vst3` y `vst/amps/BENDER DELUXE.vst3` (NAME `TW26`, UID `Tw26`) |

El 5E3 es **dos canales** (Instrument + Mic) con **volúmenes separados** que comparten
**un solo Tone**, dos triodos 12AX7 de preamp, un **cathodyne PI**, **2× 6V6GT cátodo-
común (sin NFB)** y **rectificador de tubo 5Y3GT** → cálido, woody, comprime y "florece"
temprano (el sag del 5Y3), y rompe en un grind tweed suelto cuando lo empujas.

> **Nota del reissue:** este esquemático rotula el primer tubo como **12AX7A** (no el
> 12AY7 del 5E3 vintage). El 12AX7 (mu ~100) tiene **más ganancia** que el 12AY7 (mu ~44),
> así que el reissue rompe antes/más. (El modelo DSP usa 12AY7 — ver §9.)

## Cómo leer los tubos (12AX7 / 6V6) en este esquemático
- **PLACA / ánodo** = SALIDA (a B+ por la R de carga; acopla por cap). 12AX7: pines 1/6. 6V6: pin 3.
- **REJILLA / grid** = ENTRADA; grid-leak a tierra. 12AX7: pines 2/7. 6V6 g1: pin 5, g2(pantalla): pin 4.
- **CÁTODO** = tierra por R de cátodo (+ cap bypass) — fija el bias. 12AX7: pines 3/8. 6V6: pin 8.

## 1. Entradas + V1 (12AX7A) — dos canales

```
 INSTRUMENT:  J1 (INST 1) / J2 (INST 2) ─ R3 68K / R4 68K ─► rejilla(7) V1-B · R1 1M leak⊥GND
 MIC:         J3 (MIC 1)  / J4 (MIC 2)  ─ R5 68K / R6 68K ─► rejilla(2) V1-A · R2 1M leak⊥GND
       (jacks conmutados: enchufar en el #1 pone la R de 68K en serie con el #2, estilo Fender)
            ┌──────── V1 = 12AX7A (un triodo por canal) ────────┐
            │ V1-B (INST): placa(6) RB 100K → B+ · acople C3 .1µF 400V │
            │ V1-A (MIC):  placa(1)    100K → B+ · acople C2 .1µF 400V │
            │ cátodo R7 820Ω + C1 25µF 50V (bypass) — TP11 +1.33VDC   │
            └──────────────────────────────────────────────────────────┘
```
- **V1 = 12AX7A doble triodo**, un triodo por canal: **V1-B = Instrument**, **V1-A = Mic**.
  Rejillas con grid-stopper 68K (R3/R4 INST, R5/R6 MIC) y grid-leak 1M (R1/R2). Cargas de
  placa **100K**; cátodo **R7 820Ω + C1 25µF** (bypass → ganancia plena, ~1.33V de bias).
- Cada canal acopla a su propio Volumen: **INST → C3 .1µF → R11**, **MIC → C2 .1µF → R12**.

## 2. Volúmenes (×2) + Tone stack (1 sola perilla — el tweed clásico)
```
 V1-B placa ─ C3 .1µF ─► INST VOL R11 1M (audio) ── [bright cap C4 500pF 500V a través] ─┐
 V1-A placa ─ C2 .1µF ─► MIC  VOL R12 1M (audio) ───────────────────────────────────────┤
                                                                                         ▼
                                       TONE R10 1M (audio) + C5 .0047µF 630V  ─► V2-A (§3)
```
- **Dos volúmenes interactivos** (INST R11, MIC R12 — 1M audio cada uno) que se suman a un
  **único Tone** (R10 1M + C5 .0047µF). **C4 500pF** = bright cap del Inst Vol (la chispa
  del 5E3 a volúmenes bajos). Sin pots de Bass/Mid/Treble: el 5E3 tiene **solo Tone**.

## 3. V2 (12AX7A): recovery + inversor de fase cathodyne
```
 (TONE) ─► V2-A recovery: rejilla · placa R14 100K → B+ (TP9 +165V) · cátodo R16 1K5 (+C6 25µF) · leak R15 1M
                          │
                          ─ C7 .022µF 400V ─►
 V2-B CATHODYNE (split-load PI):  placa(6) R18 56K → B+ (TP7 +204V)  ·  cátodo(8) R17 56K  (¡R placa = R cátodo!)
        placa  ─ C8 .1µF 400V ─ R21 1K5 ─► rejilla V3 (6V6 #1)   · grid-leak R19 220K
        cátodo ─ C9 .1µF 400V ─ R22 1K5 ─► rejilla V4 (6V6 #2)   · grid-leak R20 220K
```
- **V2-A** = etapa de recuperación/ganancia (placa 100K). **V2-B** = **cathodyne** (split-load):
  placa y cátodo con **R iguales (56K/56K)** generan las dos fases anti-fase para el push-pull.

## 4. Power amp: 2× 6V6GT, cátodo común (sin NFB)
```
 OP+/OP- (de la PI) ─ C8/C9 .1µF ─ R21/R22 1K5 ─► rejillas(5)
            ┌──────────── 2× 6V6GT (V3-A / V4-A) · PUSH-PULL ────────────┐
            │ pantallas(4): R24/R25 470Ω 2W → +320V   (190VAC TP18/19)    │
            │ placas(3) ─► primario del OT                                │
            │ CÁTODO COMÚN: R23 250Ω 8W + C10 25µF 50V → GND (TP6 +21.2V) │  autobias Clase AB
            └──────────────────────────────────────────────────────────────┘
            SIN realimentación global  →  el grind tweed suelto + bloom del 5Y3
```
- **2× 6V6GT** push-pull, **autobias** por **R23 250Ω 8W** compartido (+ C10 25µF), ~21V de
  cátodo. **Sin NFB global** → crudo y compresivo. Pantallas con 470Ω 2W.

## 5. Output transformer + parlante
```
 placas 6V6 ─► OT primario (CT → +370V) ─► secundario 8Ω ─► J5 SPEAKER / J6 EXTERNAL ─► 1×12 8Ω
```
- **OT** (P/N 0073304000) primario push-pull con CT a +370V; secundario **8Ω** → **1×12**
  (P/N 0073312000, cono tweed Jensen-style, oscuro/cálido). Jack extra J6 para parlante externo.

## 6. PSU — rectificador de tubo 5Y3GT
```
 MAINS ─ F1 (F2A/F1A 250V) ─ RT1 ─ C14 .1µF 275VAC ─ SW1 power ─ TX T2 (sel. 100/120/230/240V)
   HT (RED / RED-YEL center-tap) ─ D1/D2 BYD33V (diodos de protección) ─► V5 = 5Y3GT (rect. de tubo)
        ─ SW2 standby ─► +370V / C13 16µF 475V ─R27 4.7K 2W─ +320V / C12 16µF ─R26 22K 2W─ +250V / C11 16µF ─► +46V
   5V  (YELLOW · F2 T4A) ─► filamento del 5Y3GT
   6.3V(GREEN  · F3 T4A) ─► filamentos · R28/R29 100Ω 1/2W (humdinger, 3.3VAC TP3/TP4)
```
- **Rectificador de TUBO 5Y3GT** (V5) — alta resistencia interna + caps de filtro chicos
  (16µF) → **sag profundo y lento, "bloom"** = el corazón del feel 5E3. Los **D1/D2 BYD33V**
  en serie son sólo diodos de protección (no quitan el sag). **Standby SW2**.
- Escalera B+: **+370V** (placas 6V6) → **+320V** (pantallas) → **+250V** (preamp).

## 7. Controles del panel → params DSP → RS knobs

| Control real | param (`TW26ParamId`) | RS knob | nota |
|---|---|---|---|
| Instrument Volume (R11 1M + C4 500pF) | `kInstVol` | **RS Gain** | el "volumen = ganancia" del 5E3 |
| Mic Volume (R12 1M) | `kMicVol` | **RS Mid** | canal jumpeado que rellena cuerpo/medios |
| Tone (R10 1M + C5 .0047µF) | `kTone` | **RS Treble** | único control de tono real |
| Bright cap (C4 500pF) | `kBright` | RS Bright | brillo de la entrada Instrument |
| (oculto) low shelf | `kBass` | RS Bass | el 5E3 no tiene pot de graves |
| (oculto) top lift | `kPresence` | RS Pres | el 5E3 no tiene presencia |

RS expone Gain/Bass/Mid/Treble/Pres/Bright; el amp real solo tiene 2 Volúmenes + 1 Tone,
así que Bass/Pres son extensiones musicales (no pots reales).

## 8. Modelo DSP — orden de proceso (`TW26Core::process`)
Clases: `triode12AY7`/`triode12AX7`/`sixV6Pair`, `Y3CathodeBiasedSupply`, `Biquad`, etc. Orden:
1. `inputGrid` (68K/Miller) → bright input (C-equiv).
2. **V1 12AY7** (primer triodo, warm/low-mu) → Volume interactivo + **Tone** (treble bleed + shelf).
3. **Mic channel** jumpeado (RS Mid): 12AY7 paralelo más oscuro/mid-forward.
4. tweed body + Bass oculto.
5. **V2-A 12AX7** recovery.
6. **cathodyne PI + 2× 6V6** (`sixV6Pair`, bias 0.03), **5Y3 sag** (`Y3CathodeBiasedSupply`), sin NFB.
7. **OT + 1×12 tweed**: `transformerLow / speakerHp / speakerBody / speakerPresence / presenceShelf / speakerLp`.
8. **Makeup normalizado** (volumen fijo Rocksmith): `(1.50+0.40g+0.70hot)·(1 + 2/(1+36g))` → low gain a volumen pleno.

## 9. Notas / discrepancias (vs esquemático)
- **V1 = 12AX7A en el esquemático del reissue** (mu ~100), pero el DSP modela **12AY7** (vintage,
  mu ~44). El 12AX7 da **más ganancia/breakup más temprano** → a evaluar contra referencia: si
  el reissue real suena más ganoso, cambiar V1 a 12AX7 en el DSP.
- **Rectificador 5Y3GT de tubo confirmado** (no SS) → el sag/bloom del DSP es correcto.
- **Cátodo común 6V6 R23 250Ω 8W + C10 25µF** confirmado (autobias, sin NFB).
- **Solo Tone** (no Bass/Mid/Treble): RS Bass/Pres son shelves ocultos añadidos.
- Voicing del parlante (1×12 tweed oscuro) **pendiente de afinar contra referencia** (loop cerrado).
