El 5F6-A es **2 canales** (Bright + Normal) con volúmenes separados que **se mezclan** a un
**tone stack FMV** (Bass/Treble/Middle + Presence — la "Bassman stack" que copió Marshall),
**12AY7 + 2× 12AX7** de preamp, **inversor LTP (long-tailed-pair)**, **2× 5881 (6L6) bias FIJO
push-pull ~45W con NFB global**, y **rectificador GZ34**. Es más **fuerte, apretado y con
headroom** que el Deluxe 5E3 (que es cátodo-común sin NFB) — la base del rock británico.

## 0. Diagrama maestro (señal + PSU)

```
 BRIGHT  J1(1)/J2(2) ─ 68K / 68K ─► rejilla V1a · 1M leak⊥GND
 NORMAL  J3(1)/J4(2) ─ 68K / 68K ─► rejilla V1b · 1M leak⊥GND
        ┌──────── V1 = 12AY7 (un triodo por canal) ────────┐
        │ placa 100K → B+ · cátodo 820Ω + 250µF (compartido, +2.5V) │
        └──────┬──────────────────────────┬──────────────────┘
   Bright ─ .02µF ─► BRIGHT VOL 1M [bright cap .0001µF/100pF] ─ 270K ─┐
   Normal ─ .02µF ─► NORMAL VOL 1M ───────────────────────────── 270K ─┤ (mezcla)
                                                                       ▼
        ┌──────── V2 = 12AX7 (driver post-mezcla) ────────┐
        │ placa 100K → +180V · cátodo 820Ω (+1.2V)        │
        └──────┬───────────────────────────────────────────┘
               ▼  TONE STACK pasivo (FMV / "Bassman"):
   slope 100K · TREBLE 250K + cap 250pF (.00025) · BASS 1M + cap .1µF · MIDDLE 25K + cap .02µF
               ▼
        ┌──────── V3 = 12AX7 · INVERSOR LTP (long-tailed-pair) ────────┐
        │ placas 82K (5%) · cap PI 47pF · +235V · cola común            │
        └──────┬───────────────────────────┬───────────────────────────┘
   OP+ ─ .1µF ─ 220K leak ─► rejilla 5881 #1     OP- ─ .1µF ─ 220K leak ─► rejilla 5881 #2
               ▼
        ┌──────── 2× 5881 (6L6GC) · PUSH-PULL · BIAS FIJO ────────┐
        │ pantallas 470Ω 1W → B+ · rejillas ← supply de bias (−)   │  ~45W
        │ NFB GLOBAL desde el secundario del OT (+ PRESENCE)        │
        └───────────────────────┬─────────────────────────────────┘
                                ▼  OT ─► 4×10 (~2Ω, Jensen P10R)

 ── PSU ──────────────────────────────────────────────────────────────
 MAINS ─ standby ─ TX ─► HT ─► GZ34 (rect. de tubo) ─► +325V (8µF 450V) → +180V (preamp)
   supply de bias NEGATIVO (diodo + filtro) ─► rejillas 5881 vía 220K (bias fijo, ajustable)
```

## Cómo leer los tubos
- **PLACA** = salida (a B+ por R de carga). 12AX7/12AY7: pines 1/6. 5881(6L6): pin 3.
- **REJILLA** = entrada; grid-leak a tierra (o al bias −). 12AX7: 2/7. 6L6 g1: 5, g2: 4.
- **CÁTODO** = tierra por R (+ cap) = bias. 12AX7: 3/8. 6L6: 8 (a tierra → bias fijo por rejilla).

## 1. Entradas + V1 (12AY7) — dos canales
```
 BRIGHT: J1 (high) / J2 (low) ─ 68K / 68K ─► rejilla V1a · 1M leak⊥GND
 NORMAL: J3 (high) / J4 (low) ─ 68K / 68K ─► rejilla V1b · 1M leak⊥GND
            ┌─ V1 = 12AY7 ─┐  placa 100K → B+ · cátodo 820Ω + 250µF (+2.5V)
```
- **V1 = 12AY7** (mu ~44, warm/low-mu, como el 5E3): un triodo por canal (Bright/Normal).
  Grid-stopper **68K** por jack (1=high, 2=low/−6dB), grid-leak **1M**. Carga de placa **100K**,
  cátodo **820Ω + 250µF** (+2.5V).

## 2. Volúmenes (×2) + mezcla
```
 V1a placa ─ .02µF ─► BRIGHT VOL 1M [bright cap .0001µF/100pF a través] ─ 270K ─┐
 V1b placa ─ .02µF ─► NORMAL VOL 1M ─────────────────────────────────── 270K ─┴─► rejilla V2
```
- **BRIGHT VOL 1M** (con bright cap **.0001µF = 100pF** = la chispa a volumen bajo) y **NORMAL VOL 1M**.
  Los wipers se mezclan por **270K + 270K** (la suma de canales típica Fender) hacia V2.

## 3. V2 (12AX7) driver + Tone stack FMV (la "Bassman stack")
```
 (mezcla) ─► V2 12AX7: placa 100K → +180V · cátodo 820Ω (+1.2V)
            ▼
 TONE STACK pasivo:  slope 100K
   TREBLE 250K + cap 250pF (.00025)   ·   BASS 1M + cap .1µF   ·   MIDDLE 25K + cap .02µF
```
- **V2 = 12AX7** = etapa de ganancia post-mezcla (placa 100K → +180V, cátodo 820Ω). Maneja el
  **tone stack pasivo FMV**: **Treble 250K**, **Bass 1M**, **Middle 25K**, caps **250pF / .1µF / .02µF**,
  slope **100K**. Es exactamente la red que Marshall copió (JTM45) — interactiva y con scoop de medios.

## 4. V3 (12AX7) inversor LTP (long-tailed-pair)
```
 (tone stack) ─► V3 12AX7 LTP:  placas 82K (5%) · cap PI 47pF · +235V · cola/tail común
        OP+ ─ .1µF ─ 220K leak ─► rejilla 5881 #1
        OP- ─ .1µF ─ 220K leak ─► rejilla 5881 #2
```
- **V3 = 12AX7 inversor de fase LTP** (long-tailed-pair — NO cathodyne como el Deluxe). Placas
  **82K (5%)**, cap **47pF**, +235V. Genera las dos fases con más drive y headroom que un cathodyne.

## 5. Power amp: 2× 5881 (6L6) bias FIJO + NFB global
```
            ┌─ 2× 5881 (6L6GC) · PUSH-PULL ─┐
            │ pantallas 470Ω 1W → B+         │  ~45W
            │ rejillas ← bias NEGATIVO (vía 220K)  ·  cátodos a tierra │  = BIAS FIJO
            │ NFB GLOBAL desde el OT + PRESENCE (pot + cap shunt)      │
            └──────────────┬────────────────┘
                           ▼  OT ─► 4×10 (~2Ω)
```
- **2× 5881 (6L6GC)** push-pull, **bias FIJO** (rejillas a un supply negativo vía 220K; cátodos a
  tierra) → más **fuerte, apretado, headroom** que el cátodo-común del Deluxe. Pantallas 470Ω 1W.
- **NFB global** desde el secundario del OT → la **Presence** (pot + cap que sangra el NFB en agudos).
  ~**45W**. Esto + el bias fijo = el "British crunch" que copió Marshall.

## 6. PSU — rectificador GZ34
```
 MAINS ─ standby ─ TX ─► HT ─► GZ34 (rect. de TUBO, menos sag que un 5Y3) ─► +325V (8µF 450V) → +180V
   bias NEGATIVO (diodo + filtro) ─► rejillas 5881 (bias fijo ajustable)
```
- **Rectificador GZ34** (de tubo, pero más firme/menos sag que el 5Y3 del Deluxe). B+ **+325V**
  (filtro 8µF), preamp **+180V**. Supply de **bias negativo** separado para el bias fijo.

## 7. Controles → params → RS knobs
| Control real | RS knob | nota |
|---|---|---|
| Bright/Normal Volume (1M) | **RS Gain** (Bright) / **RS Mid** (Normal jumpeo) | volumen = ganancia |
| Treble (250K) | RS Treble | tone stack FMV |
| Bass (1M) | RS Bass | tone stack FMV |
| Middle (25K) | RS Mid | tone stack FMV |
| Presence | RS Pres | sangra el NFB en agudos |
| Bright cap (.0001µF) | RS Bright | chispa del canal Bright |

## 8. Diferencias clave vs el Bender Deluxe (5E3)
| | Deluxe 5E3 | **Bassman 5F6-A** |
|---|---|---|
| Power | 2× 6V6, **cátodo común**, **sin NFB** | 2× 5881, **bias fijo**, **NFB global** |
| PI | cathodyne (split-load) | **LTP (long-tailed-pair)** |
| Tono | 1 sola perilla Tone | **tone stack FMV** (Bass/Treble/Middle + Presence) |
| Rect | 5Y3 (sag profundo) | **GZ34** (más firme) |
| Carácter | woody, comprime/florece temprano | **fuerte, apretado, headroom, "British crunch"** |
